# NumPy Tutorial for Data Analytics

This repository contains a beginner-friendly NumPy tutorial in Jupyter Notebook format.

## 📚 Topics Covered

- Introduction to NumPy
- Creating Arrays
- Array Attributes
- Indexing and Slicing
- Reshaping Arrays
- Broadcasting
- Mathematical Functions
- Statistical Functions
- Copy vs View
- Stack and Concatenate
- Sorting and Filtering
- Random Module
- Practice Examples

## 🛠 Technologies Used

- Python
- NumPy
- Jupyter Notebook

## ▶️ How to Run

1. Install Python.
2. Install NumPy:

```bash
pip install numpy
```

3. Open the notebook using Jupyter Notebook or Google Colab.
