# 📊 Data Visualization with Seaborn

This repository contains my hands-on practice and learning of **Seaborn**, a Python library used for statistical data visualization.

## 📚 Topics Covered

- Introduction to Seaborn
- Line Plot (`sns.lineplot()`)
- Scatter Plot (`sns.scatterplot()`)
- Bar Plot (`sns.barplot()`)
- Histogram (`sns.histplot()`)
- Box Plot (`sns.boxplot()`)
- Count Plot (`sns.countplot()`)
- Violin Plot (`sns.violinplot()`)
- Heatmap (`sns.heatmap()`)

## 🛠️ Technologies Used

- Python
- Seaborn
- Matplotlib
- Pandas
- NumPy
- Jupyter Notebook

## 📖 What I Learned

- Creating statistical visualizations
- Understanding data distributions
- Comparing categorical data
- Finding relationships between variables
- Detecting outliers
- Visualizing correlations using heatmaps
- Customizing plots with titles, colors, legends, and styles
